How to use these buttons?

Export Selections (JSON): Downloads a user_selections.json file containing your skipped items and edited descriptions. Place this file in the same directory as report.md (the phase output directory). When you run the apply-suggestions, apply-code-fixes, or apply-task-suggestions command, it will automatically read your selections and skip the items you marked.

Copy as Text: Copies a text summary of your selections to the clipboard. You can paste this into a file or share it. This is useful for quick reference or when you prefer a simpler workflow.

Note: Your selections are automatically saved in your browser's local storage, so they persist if you refresh the page. The export is only needed when you want the orchestrator to use your selections.
